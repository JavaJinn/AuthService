# Auth Service – Spring Boot Login & Registration Template

Live Demo: (Render link will follow soon)

A portfolio-ready authentication service built with Spring Boot, Spring Security,
Thymeleaf, PostgreSQL, Flyway and Docker-ready deployment.

## Demo Login

You can create your own demo account via `/signup`.

## Features

- Secure registration flow
- BCrypt password hashing
- Spring Security login/logout
- Role-based database model
- PostgreSQL + Flyway migrations
- Environment-based configuration
- Production profile
- Health endpoint for deployment platforms
